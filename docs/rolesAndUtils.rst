.. NB: role names must be all lowercase!

.. This is how we do bolded italic text, e.g. make it a :bi:`cheeseburger`
.. role:: bi
   :class: bi

.. This is how we do bolded underline text, e.g. make it a :bu:`cheeseburger`
.. role:: bu

.. This is how we do italic underline text, e.g. make it a :iu:`cheeseburger`
.. role:: iu

.. role:: mono
   :class: mono

.. Use this role for scene description paths (e.g. /Hello/World)
.. role:: sdfpath
   :class: mono, sdfpath

.. Use this role for filenames (e.g. extras/usd/tutorials/helloWorld)
.. role:: filename
   :class: mono

.. Use this role for inline python code (e.g. UsdStage.Open)
.. role:: python
   :class: mono

.. Use this role for inline cpp code (e.g. UsdStage::Open)
.. role:: cpp
   :class: mono

.. Use this role for inline usda syntax (e.g. "def", "over")
.. role:: usda
   :class: mono

.. Use this role for code that is either ambiguous wrt to language
   (e.g. "Get()") or is not one of the languages above.
.. role:: code
   :class: mono

.. role:: underline
   :class: underline

   :class: usd-title-text sidebar-title
